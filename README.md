# Qwen2-VL VizWiz Evaluation

## Project Overview
Evaluation of Qwen2-VL-2B-Instruct on the VizWiz VQA validation set, focused on text recognition accuracy for blind and low-vision users.

## Setup
1. Clone repo
2. conda env create -f environment.yml
3. vpkg_require amd-rocm (for DARWIN cluster only if using gpumi-50 or any gpu partition with AMD-ROCm hardware, else cuda)
